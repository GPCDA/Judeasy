Bem vindo ao JudEasy!

Durante a aplicação, você perceberá que há uma navegação organizada em abas, o que indica que etapas sequenciais devem ser seguidas para uma melhor experiência com a aplicação.

O usuário deve escolher entre Treinamento e Validação para utilizar as funções da aplicação. Caso o mesmo não o faça e navegue entre as abas sem o seguimento do fluxo do sistema, o conteúdo destas abas será apenas visualizável, ou seja, todos os botões e funções das abas estarão desabilitados.

As únicas abas com botões habilitados será a "Base de Dados" e a base Histórico. A "Base de Dados" porque esta é uma maneira alternativa de se iniciar o "Treinamento". Outra forma de iniciar o Treinamento é através do botão contido nesta página inicial, conforme será explicado em "Opções.

E a aba Histórico porque é uma etapa de Visualização de informações anteriomente construídas.

Quanto às demais abas, para maiores informações, vide a "Ajuda" de cada uma delas.


Opções:


1. Treinamento:

Nesta etapa deve ser realizado o upload dos dados referente a um determinado diário, a fim de treinar um modelo de aprendizagem de máquina para reconhecer amostras deste diário na validação.

2. Validação:

Nesta etapa deve-se fazer upload de um diário e validar o mesmo através de um modelo de aprendizage de máquina que já foi treinado anteriormente.
