Treinamento


No treinamento, você deve escolher qual algoritmo de aprendizado de máquina utilizará para treinar o seu modelo, a partir de uma base de dados anteriormente escolhida.

Opções:

1. Classificadores:

Você deve escolher entre as 12 opções de classificadores disponíveis, sendo recomedado, conforme a pesquisa realizada, que o Dynamic Ensemble Selection seja o classificador escolhido.

2. Parâmetros :

Cada classificador possui seu próprio conjunto de parâmetros, são métricas que podem ser ajustadas a fim de obter melhores resultados, para saber mais sobre cada algoritmo em particular e como o ajuste de métricas funciona, visite: https://scikit-learn.org/.
